 【はじめに】
　　このたびは、「MouseGestureL.ahk」を
　　ダウンロードいただき、誠にありがとうございます。
　　本ソフトMouseGestureL.ahkは、スクリプトエンジンAutoHotkey_L上
　　で動作するマウスジェスチャースクリプトです。
　　あらゆるアプリケーションをマウスジェスチャーで操作することができます。
　　詳しい使用法は、添付のヘルプファイル
　　(~\Docs\Japanese\MouseGestureL_Help.chm)をご参照ください。

 【動作環境】
　　このソフトウェアは、v1.1.04.00以降のUnicode版AutoHotkey_L上で動作します。
　　MouseGestureL.ahkを使うにはあらかじめAutoHotkey_Lをインストールしておく
　　必要があります。(後述のZip版ではインストールの必要はありません。)
　　インストール版AutoHotkey_Lの入手先
　　http://l.autohotkey.net/AutoHotkey_L_Install.exe
　　MouseGestureL.ahkの動作環境
　　Windows 2000, XP, Vista, 7, 8
　　64bit,32bit両方をサポートしています。

 【使用方法】
　 ◆インストール
　　 解凍したファイルを任意のフォルダへコピーしてください。

　 ⇒ AutoHotkey_Lがインストールされている場合
　　　MouseGestureL.ahkを起動してください。
　　　設定画面よりスタートアップへ登録することができます。

　 ⇒ AutoHotkey_Lがインストールされていない場合
　　　同梱の「Setup.vbs」を実行してください。
　　　AutoHotkey_LをインストールすることなくMouseGestureL.ahk
　　　を使用することが出来ます。

　 ◆アンインストール
　　 アーカイブに含まれるファイルと起動後に作成されるファイルをフォルダごと
　　 削除してください。レジストリは使用していません。
　　 スタートアップへ登録した場合は、設定画面から削除できます。

　 ◆アップデート
　　 アップデート時は、アーカイブの中身を丸ごと上書きし、トレイアイコンのメニュー
　　 から「再起動」を選択してください。
　　 設定情報が記述された「MouseGestureL.ini」や、追加スクリプトが記述された
　　 「MG_User.ahk」は、配布アーカイブには含まれていないので、上書きされる心配
　　 は有りません。

 【Helpについて】
　　添付のヘルプはSpeedHelpPadで作成しました。
　　内部でsmoothscroll.jsを使っています。
　　上に戻るの操作音が気になる場合は
　　サウンドの設定>ナビゲーションの開始＞（なし）
　　に設定してください。
